XFlickr component contain backend part and some snippents for frontend

--------------------
Snippet: XPhoto
--------------------
Display photo and some info directly from flickr

Example:
[[XPhoto? &id=`640390421` &zooming=`1`]]

--------------------
Snippet: XGallery
--------------------
Display flickr photos as gallery

Example:
[[XGallery? &mode=`set` &set=`72157594531764621` &perpage=`12`]]
